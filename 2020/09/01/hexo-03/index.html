Hexo-nexT 3 生成博客并更换主题

发表于 2020-09-01 分类于 Hexo

官方文档

  • Hexo 英文文档：>>> 传送门
  • Hexo 中文文档：>>> 传送门
  • nexT 文档：>>> 传送门

初始化

  1. 选择一个空目录

    • 可以是 D:\yorkfish.github.io>
    • 也可以是 D:\hexoblog>

  2. 使用命令行工具进入该目录

  3. 本地博客访问地址：http://localhost:4000/

    • 按 Ctrl + c 停止本地服务

更换主题

  1. 在博客根目录右键 -> Git Bash Here

  2. 修改博客根目录下的 _config.yml 文件

    • 拉到底，theme: landscape -> theme: nexT

  3. 修改主题目录下的 _config.yml 文件

    • 搜索 Schemes
    • 目前 nexT 有四款样式，默认使用 scheme: Muse
    • 我想用第四种，所以注释 scheme: Muse，取消注释 scheme: Gemini

ps

  • 主题名与 themes 文件夹下的主题文件夹名要一致
  • 博客根目录与主题文件夹内都有 _config.yml，不要弄混
