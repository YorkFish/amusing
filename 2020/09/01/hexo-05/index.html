Hexo-nexT 5 修改配置文件

发表于 2020-09-01 分类于 Hexo

配置博客 _config.yml

分类与标签

  • 打开主题的 _config.yml，搜索到 menu 备用

categories

  1. D:\hexoblog>cnpm install hexo-generator-category
  2. D:\hexoblog>hexo new page categories
    • 在 source 文件夹下新建文件夹 categories
    • 在 categories 文件夹下新建文件 index.md
  3. 向 index.md 中的 YAML 内加一句 type: categories
  4. 取消 menu 下方 categories 那一行的注释

tags

  1. D:\hexoblog>cnpm install hexo-generator-tag
  2. D:\hexoblog>hexo new page tags
  3. 向 index.md 中的 YAML 内加一句 type: tags
  4. 取消 menu 下方 tags 那一行的注释

about

  1. D:\hexoblog>hexo new page about
  2. 取消 menu 下方 about 那一行的注释
  3. 直接在博客根目录下的 source\about\index.md 中写“自我介绍”即可
