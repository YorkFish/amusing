Powershell 安装 oh-my-posh

发表于 2020-10-08 分类于 Tools

Powershell

  1. 以管理员权限打开打开 Powershell

  2. 输入 Install-Module posh-git -Scope CurrentUser

  3. 系统询问是否“安装并导入 NuGet 提供程序”，输入 Y

  4. 系统询问是否“要从 PSGallery 安装模块”，输入 Y

  5. 输入 Install-Module oh-my-posh -Scope CurrentUser

  6. 系统询问是否“要从 PSGallery 安装模块”，输入 Y

  7. 输入 notepad $PROFILE

  8. 系统提示没有相关文件，并询问是否创建，点击 是(Y)

  9. 写入如下语句，保存并关闭

  10. 重启 Powershell 即可
