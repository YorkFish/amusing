选货地址

发表于 2021-01-10 更新于 2021-01-19 分类于 Exercises
  • 活动日期：2021-01-09

题目描述

  • AcWing 104
  • 简单
  • 在一条数轴上有 N 家商店，它们的坐标分别为 A1~AN。
  • 现在需要在数轴上建立一家货仓，每天清晨，从货仓到每家商店都要运送一车商品。
  • 为了提高效率，求把货仓建在何处，可以使得货仓到每家商店的距离之和最小。

输入输出

输入格式

  • 第一行输入整数 N。
  • 第二行 N 个整数 A1~AN。

输出格式

  • 输出一个整数，表示距离之和的最小值。

数据范围

  • 1 <= N <= 100000
  • 0 <= Ai <= 40000

简单证明

原理

$\left|a - x\right| + \left|b - x\right| \geq \left|a - b\right| \Rightarrow \forall x \in \left[a, b\right] 取等号$

结论

x 取中位数时，货仓到每家商店的距离之和最小

证明

\begin{aligned}
& \left|a_{1} - x\right| + \left|a_{2} - x\right| + … + \left|a_{n} - x\right|\\
= & (\left|a_{1} - x\right| + \left|a_{n} - x\right|) + (\left|a_{2} - x\right| + \left|a_{n-1} - x\right|) + …\\
= & \left|a_{1} - a_{n}\right| + \left|a_{2} - a_{n-1}\right| + …\\
\Rightarrow &
\begin{cases}
& x = a_{\frac{n+1}{2}}, & n \% 2 == 1\\
& x = a_{\frac{n}{2}} \ or\ a_{\frac{n}{2}+1}, & n \% 2 == 0
\end{cases}
\end{aligned}

补充

下面两个式子都对

证明

$\sum_{i=0}^{n-1}\left|a_{i} - a_{\frac{n}{2}}\right| = \sum_{i=0}^{n-1}\left|a_{i} - a_{\frac{i}{2}}\right|$

若 n 为奇数

\begin{aligned}
left & = (a_{\frac{n}{2}} - a_{0}) + (a_{\frac{n}{2}} - a_{1}) + … + (a_{\frac{n}{2}} - a_{\frac{n}{2}-1}) + (a_{\frac{n}{2}} - a_{\frac{n}{2}}) + (a_{\frac{n}{2}+1} - a_{\frac{n}{2}}) + … + (a_{n-2} - a_{\frac{n}{2}}) + (a_{n-1} - a_{\frac{n}{2}})\\
& = - a_{0} - a_{1} - … - a_{\frac{n}{2}-1} + a_{\frac{n}{2}+1} + a_{\frac{n}{2}+2} + … + a_{n-1}\\
right & = a_{0} + a_{1} + … + a_{n-1} - 2(a_{0} + a_{1} + … + a_{\frac{n}{2}-1}) - a_{\frac{n}{2}}, \ a_{\frac{0}{2}}=a_{\frac{1}{2}}=a_{0}\\
& = a_{\frac{n}{2}+1} + a_{\frac{n}{2}+2} + … + a_{n-1} - (a_{0} + a_{1} + … + a_{\frac{n}{2}-1})\\
\Rightarrow & left = right
\end{aligned}

若 n 为偶数

\begin{aligned}
left & = (a_{\frac{n}{2}} - a_{0}) + (a_{\frac{n}{2}} - a_{1}) + … + (a_{\frac{n}{2}} - a_{\frac{n}{2}-1}) + (a_{\frac{n}{2}} - a_{\frac{n}{2}}) + … + (a_{n-2} - a_{\frac{n}{2}}) + (a_{n-1} - a_{\frac{n}{2}})\\
& = - a_{0} - a_{1} - … - a_{\frac{n}{2}-1} + a_{\frac{n}{2}} + … + a_{n-2} + a_{n-1}\\
right & = a_{0} + a_{1} + … + a_{n-1} - 2(a_{0} + a_{1} + … + a_{\frac{n}{2}-1}), \ a_{\frac{0}{2}}=a_{\frac{1}{2}}=a_{0}\\
& = a_{\frac{n}{2}} + a_{\frac{n}{2}+1} + … + a_{n-1} - (a_{0} + a_{1} + … + a_{\frac{n}{2}-1})\\
\Rightarrow & left = right
\end{aligned}
