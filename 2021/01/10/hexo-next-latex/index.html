Hexo nexT 使用 LaTeX

发表于 2021-01-10 分类于 Hexo

配置

  • 打开 nexT 的配置文件 {path-to-blog-file}\themes\nexT\_config.yml

搜索 math

  • 打开 mathjax 开关

搜索 vendors

  • 打开 mathjax 相关的 cdn

使用

  • YAML 题头中添加 mathjax: true 即可
  • 自带的 mathjax 有些小 bug
  • 比如换行需要 4 个 \

其他做法

  • 卸载 Hexo 自带的 renderer，安装 Pandoc renderer

参考

  • Hexo 书写 LaTeX 公式时的一些问题及解决方法
  • 让 Hexo Next (v8.0.0) 支持 LaTeX 数学公式
