剪绳子

发表于 2021-01-15 更新于 2021-01-19 分类于 Exercises
  • 活动时间：2021-01-14

题目描述

  • AcWing 680
  • 简单
  • 有 N 根绳子，第 i 根绳子长度为 Li，
  • 现在需要 M 根等长的绳子，你可以对 N 根绳子进行任意裁剪（不能拼接），
  • 请你帮忙计算出这 M 根绳子最长的长度是多少。

输入输出

输入格式

  • 第一行包含 2 个正整数 N, M，表示原始绳子的数量和需求绳子的数量。
  • 第二行包含 N 个整数，其中第 i 个整数 Li 表示第 i 根绳子的长度。

输出格式

  • 输出一个数字，表示裁剪后最长的长度，保留两位小数。

数据范围

  • 1 <= N,M <= 100,000
  • 0 < Li < 109

样例

输入样例

输出样例

样例解释

  • 第一根和第三根分别裁剪出一根 2.50 长度的绳子，
  • 第二根剪成 2 根 2.50 长度的绳子，刚好 4 根。

默认代码

  • 无

Solution

  1. 设裁剪后的长度为 x
  2. 计算给定的绳子能被裁剪成几段长为 x 的绳子，不妨记作 y 段
  3. 判断 M 与 y 的关系
    • 若 M <= y，说明可以再裁长点，此时改变左端点
    • 若 y < M，说明裁长了，此时改变右端点
