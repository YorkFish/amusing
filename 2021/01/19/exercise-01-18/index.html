翻硬币

发表于 2021-01-19 分类于 Exercises
  • 活动时间：2021-01-18

题目描述

  • AcWing 1208
  • 简单
  • 小明正在玩一个“翻硬币”的游戏。
  • 桌上放着排成一排的若干硬币。我们用 * 表示正面，用 o 表示反面（是小写字母，不是零）。
  • 比如，可能情形是：**oo***oooo
  • 如果同时翻转左边的两个硬币，则变为：oooo***oooo
  • 现在小明的问题是：如果已知了初始状态和要达到的目标状态，每次只能同时翻转相邻的两个硬币，那么对特定的局面，最少要翻动多少次呢？
  • 我们约定：把翻动相邻的两个硬币叫做一步操作。

输入输出

输入格式

  • 两行等长的字符串，分别表示初始状态和要达到的目标状态。

输出格式

  • 一个整数，表示最小操作步数

数据范围

  • 输入字符串的长度均不超过 100。
  • 数据保证答案一定有解。

Solution

  • 对于一排足够长的硬币，若 n 个硬币只翻一次，共有 n-1 种翻法
  • 又因为这 n-1 种方法中任一种翻法使用两次等于没有使用，所以每种翻法最多会被使用一次
  • 若第一个硬币需要翻转，则只能翻头两个硬币，接下来就是解决子问题：翻转第二个及之后的共 n-1 个硬币
  • 子问题相当于把上一步重新做一遍
  • 因为题目保证一定有解，所以不必担心最后一个硬币
