滑雪场设计

发表于 2021-01-26 分类于 Exercises
  • 活动时间：2021-01-25

题目描述

  • AcWing 1353
  • 简单
  • 农夫约翰的农场上有 N 个山丘，每座山的高度都是整数。
  • 在冬天，约翰经常在这些山上举办滑雪训练营。
  • 不幸的是，从明年开始，国家将实行一个关于滑雪场的新税法。
  • 如果滑雪场的最高峰与最低峰的高度差大于 17，国家就要收税。
  • 为了避免纳税，约翰决定对这些山峰的高度进行修整。
  • 已知，增加或减少一座山峰 x 单位的高度，需要花费 x2 的金钱。
  • 约翰只愿意改变整数单位的高度。
  • 请问，约翰最少需要花费多少钱，才能够使得最高峰与最低峰的高度差不大于 17。

输入输出

输入格式

  • 第一行包含整数 N。
  • 接下来 N 行，每行包含一个整数，表示一座山的高度。

输出格式

  • 输出一个整数，表示最少花费的金钱。

数据范围

  • 1 <= N <= 1000
  • 数据保证，每座山的初始高度都在 0∼100 之间。

样例解释

  • 最佳方案为，将高度为 1 的山峰，增加 3 个单位高度，将高度为 24 的山峰，减少 3 个单位高度。

默认代码

  • 无

Solution

  • 可以用反证法证明最终解必在边界 [0, 100] 内
    • 如果有解在边界外，不妨设那些解小于 0，如果把那些解都移到 0，由于其原值（边界内）到 0 的距离小于到解的距离，所以移到 0 的代价更小，这样就与解矛盾，所以解必在边界内
  • 在边界内枚举所有跨度为 17 的范围，求出这些范围的代价，取最小值

补充

  • 如果约翰不只愿意改变整数单位的高度，那么上面的做法会失效
  • 反例：两个山峰 0, 18
    • 原来的解是 0 增高到 1 或 18 降低到 17，花费是 1
    • 现在只将 0 增高到 0.5，将 18 降低到 17.5，花费就小于 1
