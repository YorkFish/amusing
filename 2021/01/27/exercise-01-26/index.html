整数集合划分

发表于 2021-01-27 分类于 Exercises
  • 活动时间：2021-01-26

题目描述

  • AcWing 1603
  • 简单
  • 给定一个包含 N 个正整数的集合，请你将它划分为两个集合 A1 和 A2，其中 A1 包含 n1 个元素，A2 包含 n2 个元素。
  • 集合中可以包含相同元素。
  • 用 S1 表示集合 A1 内所有元素之和，S2 表示集合 A2 内所有元素之和。
  • 请你妥善划分，使得 $\left|n1 − n2\right|$ 尽可能小，并在此基础上 $\left|S1 − S2\right|$ 尽可能大。

输入输出

输入格式

  • 第一行包含整数 N。
  • 第二行包含 N 个正整数。

输出格式

  • 在一行中输出 $\left|n1 − n2\right|$ 和 $\left|S1 − S2\right|$，两数之间空格隔开。

数据范围

  • $2 \leq N \leq 10^5$
  • 保证集合中各元素以及所有元素之和小于 $2^{31}$。

Solution

  • 均分能使 $\left|n1 − n2\right|$ 尽可能小
    • 若 N 为偶数，$\left|n1 − n2\right| = 0$
    • 若 N 为奇数，$\left|n1 − n2\right| = 1$
  • 不妨设最终 $S1 \leq S2$，可以用反证法证明 $\forall a1 \in A1$, $\forall a2 \in A2$ 都有 $a1 \leq a2$
    • 假设 $a1 > a2$
    • 交换两数阵营，可以使 S1 更小，S2 更大，从而 $\left|S1 − S2\right|$ 更大
    • 如此 S1 与 S2 便不是最终解，矛盾
  • 将 N 个数升序排序，前一半是 A1，后一半是 A2
    • 若 N 为偶数，对半分即可
    • 若 N 为奇数，对半分会多一个，显然，A2 多一个数能使 $\left|S1 − S2\right|$ 更大
