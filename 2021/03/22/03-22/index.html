85 不用加减乘除做加法

发表于 2021-03-22 分类于 AcWing

题目

写一个函数，求两个整数之和，要求在函数体内不得使用 ＋、－、×、÷ 四则运算符号。

样例

Default

Solution

  • 不进位的加法就是“一个是1另一个是0 => 取到1”，正好可以用异或表示
  • 进位的加法就是“两个都是1 => 取到10”，正好可以用相与表示，因为进位，所以需要左移一位
