19 二叉树的下一个节点

发表于 2021-03-28 更新于 2021-04-07 分类于 AcWing

题目

给定一棵二叉树的其中一个节点，请找出中序遍历序列的下一个节点。

注意

  • 如果给定的节点是中序遍历序列的最后一个，则返回空节点;
  • 二叉树一定不为空，且给定的节点一定不是空节点；

样例

Default Code

Solution

  1. 如果有右子树，那么后继是右子树“最左下角”的结点
  2. 如果没有右子树，那么往上找，目标是“右上”，若找到的是“左上”，则一直往上找
    • 因为有 father 指针，所以找父结点是 O(h) 的
    • 如果所有点都求一次后继，总时间复杂度是 O(n) 的
      • 步骤 1，从上往下，所有点会遍厉一次
      • 步骤 2，从下往上，所有点又遍厉一次
