整数集合划分

发表于 2021-01-27 分类于 Exercises
  • 活动时间：2021-01-26

题目描述

  • AcWing 1603
  • 简单
  • 给定一个包含 N 个正整数的集合，请你将它划分为两个集合 A1 和 A2，其中 A1 包含 n1 个元素，A2 包含 n2 个元素。
  • 集合中可以包含相同元素。
  • 用 S1 表示集合 A1 内所有元素之和，S2 表示集合 A2 内所有元素之和。
  • 请你妥善划分，使得 $\left|n1 − n2\right|$ 尽可能小，并在此基础上 $\left|S1 − S2\right|$ 尽可能大。

滑雪场设计

发表于 2021-01-26 分类于 Exercises
  • 活动时间：2021-01-25

题目描述

  • AcWing 1353
  • 简单
  • 农夫约翰的农场上有 N 个山丘，每座山的高度都是整数。
  • 在冬天，约翰经常在这些山上举办滑雪训练营。
  • 不幸的是，从明年开始，国家将实行一个关于滑雪场的新税法。
  • 如果滑雪场的最高峰与最低峰的高度差大于 17，国家就要收税。
  • 为了避免纳税，约翰决定对这些山峰的高度进行修整。
  • 已知，增加或减少一座山峰 x 单位的高度，需要花费 x2 的金钱。
  • 约翰只愿意改变整数单位的高度。
  • 请问，约翰最少需要花费多少钱，才能够使得最高峰与最低峰的高度差不大于 17。

题目描述

  • AcWing 1381
  • 简单
  • N 的阶乘（记作 N!）是指从 1 到 N（包括 1 和 N）的所有整数的乘积。
  • 阶乘运算的结果往往都非常的大。
  • 现在，给定数字 N，请你求出 N! 的最右边的非零数字是多少。
  • 例如 5! = 1x2x3x4x5 = 120，所以 5! 的最右边的非零数字是 2。
